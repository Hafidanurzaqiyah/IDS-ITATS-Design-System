Buttons

Button digunakan untuk menginisialisasi suatu tindakan. Button labels mengungkapkan tindakan apa yang akan terjadi ketika pengguna berinteraksi dengannya.

Size Small Add this class .btn-sm 
<div class = "row"
          <button type = "button" class = "btn btn-success btn-sm">Login</button>
          <button type = "button" class = "btn btn-success btn-sm">Submit</button>
          <button type = "button" class = "btn btn-success btn-sm">Edit</button>
          <button type = "button" class = "btn btn-success btn-sm">Delete</button>
          <button type = "button" class = "btn btn-success btn-sm">Save</button>
          </div>
