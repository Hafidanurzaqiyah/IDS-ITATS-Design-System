Typography

Typography berguna untuk membuat hirarki yang jelas agar pembaca dapat membaca senyaman mungkin.

Font

Penggunaan font sangat penting dan harus diperhatikan. Banyak framework yang menyediakan font secara gratis yang dapat kita gunakan, seperti google font.

Font-family : Properti font-family digunakan untuk menentukan font dari sebuah elemen.

body {

            font-family: serif;
            
}

PT Serif

PT Serif adalah jenis huruf serif transisi dengan terminal humanistik. Ini dirancang untuk digunakan bersama dengan PT Sans, dan diselaraskan di seluruh metrik, proporsi, bobot, dan desain.


Font weight

Font-weight digunakan untuk membedakan atau menekankan hierarki konten.

  • 300 — Light / Light Serif
  • 400 — Normal / Normal Serif
  • 500 — Medium / Medium Serif
  • 600 — Semibold / Semibold Serif
  • 800 — Bold / Bold Serif
  • 900 — Bolder / Bolder Serif
