Icon

Icon digunakan sebagai simbol untuk mewakili suatu objek atau tindakan. Icon juga digunakan supaya hemat dalam memberikan kejelasan suatu objek atau tidakkan.

Icon

Beberapa icon yang dapat digunakan dalam sistem kami. Ada beberapa framework css yang menyediakan kumpulan icon secara gratis atau perbayar. Sebagai acuan kami menggunakan beberapa framework tersebut yaitu :


Font Awesome CDN 4.7.0

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">

BoxIcon CDN

<link href='https://unpkg.com/boxicons@2.0.7/css/boxicons.min.css' rel='stylesheet'>
