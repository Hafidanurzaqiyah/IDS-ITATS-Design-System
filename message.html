Message

Pesan memungkinkan untuk mengkomunikasikan informasi kepada pengguna. Pesan dapat bertindak sebagai pemberitahuan atau jenis pesan peringatan lainnya.

Success message

Terdapat pesan ketika sukses dalam input

This is a success alert—check it out! 
<div class="alert alert-success mr-auto ml-auto component" role="alert">
              This is a success alert—check it out!
              </div>

Warning message

Terdapat pesan untuk peringatan pada user

This is a warning alert—check it out! 
 <div class="alert alert-warning mr-auto ml-auto component" role="alert">
                  This is a warning alert—check it out!
                  </div>

Danger message

Pesan bahaya untuk pemberitahuan pada user ketika salah langkah

This is a danger alert—check it out! 
<div class="alert alert-danger mr-auto ml-auto component" role="alert">
                      This is a danger alert—check it out!
                      </div>
